## 项目介绍
基于小程序的单词记忆小程序管理系统【含报告文档】：前端 小程序、jsp、jquery，后端 springmvc、spring、mybatis；角色分为管理员、用户；集成单词列表、学习模范、我的收藏、文章管理、评价管理等功能于一体的系统。
